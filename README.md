# Order Management System — Backend

Backend system for laboratory order management built with **TypeScript and MongoDB**, focusing on business rules, clean architecture, and automated tests.

## Features
- JWT authentication (register/login)
- Orders CRUD with pagination and filters
- Controlled state flow (CREATED → ANALYSIS → COMPLETED)
- Business rule validations
- Unit and integration tests (Vitest)

## Tech Stack
Node.js • TypeScript • Express • MongoDB • JWT • Vitest

## Getting Started
npm install  
npm run dev  

## Tests
npm test
